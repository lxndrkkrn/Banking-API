# Core Banking API
Тестовый проект api банка v1.2.2.

### Стек:
- Java 21, Spring Boot 4
- Spring Data JPA, PostgreSQL
- Lombok, Swagger UI

### Функционал:
- Создание пользователей и счетов.
- Транзакционные переводы денег между счетами.
- История транзакций с фильтрацией.
- Глобальная обработка ошибок через `@ControllerAdvice`.
- Логирование через `@Slf4j`.
- Защита баз данных через `DTO`.

### Нововведения:
- Добавлены некоторые тесты.
- Обновлены методы Создания/Удаления пользователей/счетов.
- Исправлена критическая ошибка при переводе.
- Мелкие изменения.

### Микроизменения:
- Доработка теста для проверки успешной транзакции.
