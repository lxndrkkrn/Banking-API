# Core Banking API
Тестовый проект api банка v1.4.0.

### Стек:
- Java 21, Spring Boot 4
- Spring Data JPA, PostgreSQL, Spring Security (ALPHA)
- Lombok, Swagger UI

### Функционал:
- Создание пользователей и счетов.
- Транзакционные переводы денег между счетами.
- История транзакций с фильтрацией.
- Глобальная обработка ошибок через `@ControllerAdvice`.
- Логирование через `@Slf4j`.
- Защита баз данных через `DTO`.
- JUnit тесты.

### Нововведения:
- Тесты создания пользователя.
- Ошибка, возникающая при незаполнении обязательных полей.
- Общая ошибка 422.

### Микроизменения:
- Доделана система создания пользователя.
- Исправлены некоторые баги.
